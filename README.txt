How to install (developer mode):
1. Place the 'FavTheme' folder on your machine.
2. Open Chrome, go to chrome://extensions/
3. Enable Developer mode (top right)
4. Click 'Load unpacked' and select the FavTheme folder
5. Click the FavTheme icon and use the popup controls

Features:
- Force dark theme using injected CSS
- Brightness slider to tweak brightness
- Exclude list to skip certain sites
- Auto-schedule to enable dark mode during night hours
